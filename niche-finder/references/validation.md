# 四步验证流程

拿到 Top 候选后，**不要直接下"做"的结论**——先走四步。整套流程约 30 分钟一个词，但能省下 1 周无效建站。

## Step 1: Semrush / Ahrefs 数据验证

**查什么**：
- Volume（搜索量）
- KD（关键词难度）
- 意图（Intent：Informational / Navigational / Commercial / Transactional）

**通过条件（新手档）**：

| 指标 | 通过 | 警告 | 硬否决 |
|---|---|---|---|
| KD | 0 – 29 | 30 – 50 | > 50 |
| Volume | 500 – 10,000 | 200 – 500 | < 200 |
| 意图 | Transactional / Commercial | Informational | 纯导航词 |

**边界案例**：
- KD = 35 但 CPC > $3 → 可考虑，但需要差异化内容
- Volume = 100 但 CPC > $5（高价词） → 可以，小而美
- 信息型意图（"what is XXX"）+ CPC 低 → 一般不做工具站

## Step 2: Google Trends 趋势验证

**网址**：[trends.google.com](https://trends.google.com)

**查什么**：
- **12 个月曲线**：是否平稳 / 上升 / 下降？
- **5 年曲线**：长期趋势（避免昙花一现）
- **国家分布**：目标国家是否有量

**通过条件**：

| 曲线特征 | 判定 |
|---|---|
| 全年稳定 | ✅ 通过 |
| 持续上升 | ✅ 优选 |
| 持续下降（>30% 年跌幅） | ❌ 不做 |
| 强季节性（<3 个月有量） | ❌ 不做 |
| 爆发型新兴词（曲线陡峭） | ⚠️ 速度优势窗口，但要验证真假 |

**举例**：
- "Christmas Card Generator" → 12 月一个月，其他月几乎为 0 → 不做
- "AI Headshot Generator" → 2023 年后持续上升 → 做
- "Fax Machine Review" → 多年持续下降 → 不做

## Step 3: Google SERP 竞争验证（最关键）

**怎么做**：直接去 Google 搜你的关键词（建议开无痕 + 目标国家 VPN），看**第一页 10 条**。

**通过条件**：

| SERP 首页特征 | 判定 |
|---|---|
| 至少 3-5 条是不知名小站/博客（DA < 30） | ✅ 通过——机会在 |
| 1-2 条大站 + 其余小站 | ✅ 可做 |
| 8+ 条大站（Wikipedia / Amazon / GitHub / 官方品牌） | ❌ 不做 |
| 全是聚合/评测大站（如 Forbes / NYTimes） | ❌ 不做 |
| SERP 被 Reddit / Quora 主导 | ⚠️ 信息型意图，不适合工具站 |

**核心问题问自己**：
> 如果我的网站出现在这个搜索结果里，用户会点进来吗？

用户点击概率低 = 做了也白做。

**额外信号**：
- SERP 顶部有大量广告 → 变现强，+1
- SERP 有 Featured Snippet / People Also Ask → 有内容型流量可抢
- SERP 前三是 Reddit 帖子 → 用户信任 UGC，工具站要特别强理由才能切入

## Step 4: CPC 变现验证

**查什么**：CPC（Cost Per Click，广告主为这个词支付的点击单价）

**通过条件**：

| CPC | 变现前景 | 判定 |
|---|---|---|
| ≥ $2 | 强商业意图，Adsense + 会员都能走 | ✅ 优选 |
| $0.5 – $2 | 正常工具站区间 | ✅ 通过 |
| $0.1 – $0.5 | 变现能力一般，能做但收益低 | ⚠️ 谨慎 |
| < $0.1 | 广告主不愿付费，难变现 | ❌ 不做 |

**高 CPC 行业参考**：金融、法律、医疗、B2B SaaS、保险、房产（但 KD 普遍高）

**中等 CPC 的新手最佳区间**：工具类（Generator / Converter / Calculator），CPC $0.3 – $3，KD 可控。

## 综合决策表

| Step 1 | Step 2 | Step 3 | Step 4 | 决策 |
|---|---|---|---|---|
| ✅ | ✅ | ✅ | ✅ | **做** — 可直接建站 |
| ✅ | ⚠️ | ✅ | ✅ | 做，但关注趋势 |
| ✅ | ✅ | ⚠️ | ✅ | 做，但内容差异化要强 |
| ✅ | ✅ | ✅ | ⚠️ | 可做信息站，不适合工具站 |
| ❌ | any | any | any | **不做** — Step 1 硬否决 |
| any | ❌ | any | any | **不做** — 季节词/下降词 |
| any | any | ❌ | any | **不做** — SERP 打不过 |
| any | any | any | ❌ | **不做** — 无法变现 |

## 时间预算

一个候选的完整验证约 **30 分钟**：

- Step 1：5 分钟（Semrush 查表）
- Step 2：5 分钟（Trends 看曲线）
- Step 3：10 分钟（SERP 人工分析）
- Step 4：5 分钟（CPC 查表 + 变现推演）
- 记录决策：5 分钟

**10 个候选 = 半天**。走完比"拍脑袋直接建站然后做一周发现没流量"划算得多。
